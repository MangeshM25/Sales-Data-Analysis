📊 Sales Performance Analytics Dashboard | Power BI
Overview

This Power BI project provides an interactive and data-driven view of sales performance, helping businesses track key metrics, identify trends, and make informed decisions. The dashboard combines advanced data modeling, DAX calculations, and interactive visualizations to deliver actionable insights.

Key Features
Top & Bottom 5 Products by Sales, Profit, and Quantity Sold
Sales Trend Analysis (Daily, Monthly, Quarterly, and Yearly)
Sales vs Profit Correlation Analysis
Dynamic Period Comparison using user-selected date ranges
Discount Category Performance Analysis
Total Orders, Sales, Profit, Net Sales, and Quantity KPIs
Order-Level Analysis with interactive filtering (Product, Date, Customer, Promotion, Category, etc.)
Regional and Category-wise Sales Performance


Skills Demonstrated
Power BI Dashboard Development
Data Modeling (Star Schema)
Power Query (ETL & Data Transformation)
DAX Measures & Time Intelligence
Interactive Reports, Drill-through, and Dynamic Filters
Business Intelligence & Data Visualization
